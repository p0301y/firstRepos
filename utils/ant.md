## 什么是webview
webView（网络视图）能够加载显示网页，可以将其视为一个浏览器，它可以使用webkit渲染引擎加载显示
网页，所以使用h5开发需要结合webView；有andriod webView和ios webView
```
//如何判断设备的类型
var sysPlatform
var ua = navigator.userAgent.toLowerCase()
if(/iphone|ipad|ipod/.test(ua)){
    sysPlatform = "IOS"
}else if(/andriod/.test(ua)){
    sysPlatform = "ANDRIOD"
}
```
